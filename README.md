📊 Institutional Portfolio Risk & Performance Dashboard

## 📌 Overview
An interactive Power BI dashboard analyzing 5-year performance of Malaysian equities against KLCI benchmark. Built for investment analysis and portfolio monitoring.

## 📈 Dashboard Pages

| Page | Description |
| **Cover** | Project overview and instructions |
| **Performance** | KPI cards + stock performance vs KLCI |
| **Risk** | Scatter plot (risk vs return) + treemap + matrix |
| **Insights** | Executive summary and key takeaways |

## 📊 Key Metrics Calculated
- **CAGR**: Compound Annual Growth Rate (average yearly return)
- **Volatility**: Annualized price fluctuation (risk measure)
- **Sharpe Ratio**: Return per unit of risk (higher = better)
- **Daily Returns**: Day-over-day percentage changes

## 🔍 Key Findings
- Portfolio returned **-2.0% CAGR** with **64% volatility**
- Only **2 of 6 stocks** had positive Sharpe (Maybank, TNB)
- Best performer: **Maybank** (+11% CAGR)
- Worst performer: **Genting** (-28% CAGR)
- Market (KLCI) returned just **0.94%** over 5 years
